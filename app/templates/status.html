Operator Guide

  • Healthy: All systems operational. No action needed.
  • Degraded: Some providers slow or catalog has minor issues. Review and monitor.
  • Unhealthy: Critical provider down or significant catalog issues. Immediate attention required.

Thresholds:

  • Unhealthy if >10% tracks have missing files
  • Degraded if any tracks have missing files or >20% missing duration
  • Run reindex to update duration metadata for all tracks
